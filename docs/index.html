Otra Ruta

Otra Ruta, es una aplicación financiada por el Ministerio de las Culturas, las Artes y el Patrimonio.

Cómo instalar la app en Android

  1. Descarga el archivo APK desde este sitio.
  2. Abre el archivo descargado desde la barra de notificaciones o el gestor de archivos.
  3. Cuando el sistema lo indique, toca Configuración y habilita Permitir desde esta fuente.
  4. Regresa a la pantalla de instalación y toca Instalar.
  5. Una vez finalizado el proceso, abre la aplicación normalmente.
Esta aplicación es segura para instalar. Pronto disponible en App Store y Play Store.
