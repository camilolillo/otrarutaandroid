Otra Ruta

Otra Ruta es una aplicación cultural financiada por el Ministerio de las Culturas, las Artes y el Patrimonio de Chile, orientada a la puesta en valor del territorio y el patrimonio local.

Cómo instalar la app en Android

1. Descarga el archivo APK desde este sitio.

2. Abre el archivo desde la barra de notificaciones o el gestor de archivos.

3. Cuando el sistema lo indique, habilita Permitir desde esta fuente.

4. Regresa a la instalación y toca Instalar.

5. Una vez finalizado el proceso, abre la aplicación normalmente.

Política de Privacidad

Última actualización: 2026

Otra Ruta es una aplicación de carácter cultural y educativo, de acceso gratuito. La aplicación no solicita registro de usuario ni recopila información personal identificable.

La app no almacena, procesa ni comparte datos personales. No utiliza servicios de terceros con fines de análisis, publicidad o seguimiento.

El uso de la ubicación del dispositivo es exclusivamente para mostrar contenidos culturales geolocalizados en el mapa. Esta información no se almacena ni se comparte.

Cualquier modificación a esta política será informada mediante una actualización de la aplicación.
